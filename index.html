KEGG Pathway XML File (KGML) Reader for Cytoscape 2.x

Introduction

Cytoscape is a widely used bioinformatics software platform for integrating, analyzing, and visualizing biological network data sets. KGMLReader is a plugin for importing KGML files to Cytoscape.

About KEGG

KEGG Pathway database is one of the most comprehensive databases of human-curated biological pathways. KGML (KEGG Markup Language) is the file format to represent KEGG Pathway data files in XML.

This is one of the Google Summer of Code 2010 projects.
